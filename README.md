# Calculator

## Installition

ENGINE: **node >= 8.0.0**
From root dirictory

```bash
npm install
```

or, if using *yarn*


```bash
yarn install
```

## Build

### Development

```bash
npm run build
```

or, if using *yarn*


```bash
yarn build
```


### Production

```bash
npm run build:prod
```

or, if using *yarn*


```bash
yarn build:prod
```

## Start develop

```bash
npm run start
```

or, if using *yarn*


```bash
yarn start
```
